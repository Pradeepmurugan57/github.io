⭐ Featured Project

Data-to-Dashboards 360 — Retail Analytics Warehouse

Architected a full retail analytics warehouse using star/snowflake schemas with a central FactOrders table and conformed dimensions (Date, Customer, Product, Store, Loyalty). Generated realistic synthetic datasets with Python for 100 stores (~10k–100k rows/day), loaded via SnowSQL & COPY INTO, with continuous ELT via AWS S3 + Snowpipe event triggers. Delivered Power BI dashboards with drill-downs by region, store, and product — KPIs include Revenue, AOV, Discount/Shipping impact, and YoY comparisons.
